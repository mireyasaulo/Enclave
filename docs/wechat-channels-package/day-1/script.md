## TTS 提示词

```
角色：24-28 岁年轻女性，温和场景
语气：平稳、有重量、略带感慨
情感：第 1 段平稳叙述（4 个排比"打开...是..."要等距）
       第 2 段轻微转折（"所以我做了一个东西"温度稍升）
       第 3 段稍带温度（"他们彼此认识、偶尔吵架"是有画面感的）
```

约 130 字 · 实际成片 60 秒（含静默与画面镜头时间）
