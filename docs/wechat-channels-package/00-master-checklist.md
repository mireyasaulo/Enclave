# 每条视频出片自检清单（视频号专属）

每条视频成品交付前，逐项确认：

## 内容正确性

- [ ] **聊天/朋友圈/群聊文本与 `day-N/data/data.json` 100% 一致**（一字不差）
- [ ] 角色名字、头像、关系标签与 data.json 一致
- [ ] 时间戳合理（画面里的时间要么是真实数字要么不显示，不能矛盾）
- [ ] 数字单位准确

## 时长（视频号专属阈值）

- [ ] 视频时长在 `shot-list.md` 给定范围内（±3 秒），落在 60-90s 区间
- [ ] **首 5 秒内**出现 `script.md` 标注的钩子原文，**字幕必须呈现**（视频号比抖音宽松 2 秒）

## 视觉规范

- [ ] 分辨率 = 1080×1920，30fps，H.264
- [ ] 全程字幕硬烧录（不要外挂 SRT）
- [ ] 字幕字号 ≥ 60px、白字 + 2px 黑描边
- [ ] 录屏画面**未变形、未拉伸**（保持原比例，背景用模糊扩展填充）
- [ ] 录屏画面有 12px 圆角 + 2px 描边
- [ ] 关键内容不出现在**视频号 UI 安全区**（顶 180 / 底 320 / 右 160 / 左下 360×180）
- [ ] 字幕区在 Y=1100-1480（已往上移 100px，避开扩展链接卡片）

## 字幕

- [ ] 字幕**先于或同步于**配音出现
- [ ] 字幕停留至少 800ms 后再消失
- [ ] 单行不超过 14 中文字
- [ ] 强调字（≤1 句/条视频）字号放大到 88px + 颜色 `#f97316`

## 音频

- [ ] 配音是统一音色（7 条同一人声 / 同一 TTS 配置）
- [ ] 配音 0dB，BGM -22 至 -18dB
- [ ] 关键台词处 BGM 进一步压低（-26dB）
- [ ] 片尾最后 1 秒 BGM 切断（不要淡出）
- [ ] BGM 来自 CC0 / Pixabay Music / 视频号商用库
- [ ] 视频号节奏 240-260 字/分（比抖音慢）

## 封面

- [ ] 封面 1080×1440 JPG（不是 1080×1920，封面比例不一样）
- [ ] 与 `cover-spec.md` 描述一致
- [ ] 封面大字字号 ≥ 110px、白字 + 黑描边
- [ ] 封面包含 1 个核心视觉元素 + 1 句钩子文案，**不超 2 句**

## 风险词扫描

- [ ] 视频中**无**「ChatGPT」「GPT」「AI 女友」字样
- [ ] 视频中**无**其他平台 logo（抖音/小红书/B 站/X/TikTok）— 仿真截图需打码
- [ ] 视频中**无** github.com / 任何 URL（用"开源"绕开）
- [ ] 视频中**无**真实手机号、邮箱、IP
- [ ] 视频中**无**「家人们」「姐妹们」「点赞投币」字样

## 视频号专属（关键）

- [ ] **视频号底部"扩展链接"已配置**，挂的是 `publish-meta.md` 指定的那篇公众号文章
- [ ] 公众号文章已在 T-1 天提前发表（不能视频先于文章）
- [ ] 视频时长在 60-90s 区间（视频号最佳完播）

## 发布前

- [ ] `publish-text.txt` 文案字数 ≤ 100 字（视频号文案区比抖音小）
- [ ] `pinned-comment.txt` 是一条**自然语气的"用户视角"评论**（不要明显宣传腔）
- [ ] 标签数量 5–8 个（视频号标签效用比抖音低，6 个左右最佳）
- [ ] 标签第一个必须与视频主题强相关（不是 `#ai` 这种泛标签）

## 结构性自检

- [ ] 前 5 秒有强钩子（情绪/反差/悬念三选一）— 视频号比抖音宽松 2 秒
- [ ] 中段有 1 个"密度爆发点"（信息/笑点/反转）
- [ ] 末尾有"收束动作"（不是逐渐淡出，是干脆的结尾字幕 + BGM 切断）
- [ ] **整体节奏有呼吸感**（不像抖音那么挤）— 这是视频号最关键的差异

---

## 不通过项处置

任何一项 ❌ → **不交付**，回到 shot-list.md 修订。

## 例外情况

- 某项无法确认（如 BGM 是否商用、字数是否超）→ **保守处理**，不要赌
- 数据 JSON 与脚本冲突 → **以数据 JSON 为准**，反馈到 `output/questions.md`，UP 主修改后重新生产

## 13 条总览（速查）

1. 文本与 data.json 一致
2. 时长 60-90s
3. 首 5 秒有钩子+字幕
4. 1080×1920 / 30fps / H.264
5. 字幕硬烧 / Y=1100-1480
6. 视频号 UI 安全区不被遮挡
7. 配音统一音色 / 240-260 字/分
8. BGM CC0+片尾切断
9. 封面 1080×1440 + 大字 110px+
10. 无风险词 / 无其他平台 logo
11. **底部扩展链接挂对的公众号文章**
12. 公众号文章 T-1 天已发
13. 节奏有呼吸感（不像抖音那么挤）
