## 跨条联动（视频号专属）

### 1. 视频号主页简介（发布 D1 前调整）

视频号主页简介改为：
> 一个属于你的 AI 虚拟世界｜评论区告诉你怎么玩

### 2. 每条挂公众号文章

每条视频底部"扩展链接"挂当天对应主题的公众号 7 天图文（见 `00-public-account-link-map.md`）。

这是**视频号专属能力**（抖音没有）。视频负责情感钩子，公众号承接深度种草，形成 **视频号 → 公众号 → 官网体验** 三段闭环。

### 3. 评论区品牌词埋点

每条视频发布后 30 分钟内，**必须**让小号在评论区出现一次「**隐界**」品牌词。前 3 楼至少 1 楼。

具体话术见每条 `publish-meta.md` 的"评论区第一条"+"发布后 SOP"。

### 4. D7 后的"作品集墙"

D7 发布后 2 小时内：
1. 把 D1–D7 全部按播放量排序置顶
2. 视频号主页简介改为：「7 条故事，1 个世界。完整看完，你会想试一下。」

## 数据目标（每条视频）

| 指标 | 阈值（视频号比抖音稍高） |
|------|----|
| 完播率 | ≥ 40%（D1） / ≥ 45%（D2-D6） / ≥ 50%（D7） |
| 5 秒留存 | ≥ 55% |
| 互动率（评赞比） | ≥ 3.5% |
| 转发率 | ≥ 1.2%（视频号专属指标，决定能否进入"朋友点赞"流量池） |
| 24h 涨粉 | ≥ 80 粉 |
| 24h 播放 | ≥ 6k |
| 公众号文章点击 | ≥ 200 次/条（"扩展链接"点击数） |

不达标 → 优化下一条钩子。
