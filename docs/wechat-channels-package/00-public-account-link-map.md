# 视频号"扩展链接"挂公众号文章 · 配置说明 + 7 条对照表

## 什么是视频号"扩展链接"

视频号原生功能：发布视频时可在底部挂一个"扩展链接"卡片，**只能挂同主体的公众号文章 / 视频号小店商品 / 微信小程序**。

本包统一挂"公众号文章"。这是抖音没有的能力，是视频号 → 公众号 → 官网 转化漏斗的核心环节。

---

## 配置步骤（视频号助手 / 创作者中心）

1. 视频号创作者中心 → 发表视频
2. 视频上传完成后，在"发表设置"页面下方找到 **"扩展链接"** 选项
3. 选择 **"公众号文章"**
4. 在搜索框输入文章标题关键词，选择对应那篇
5. 文章会显示为视频底部的卡片：📄 标题 + 公众号名称 + 阅读数

⚠️ **前置条件**：
- 视频号主体必须和公众号主体一致（同一个微信号注册或已认证关联）
- 公众号文章必须在 T-1 天就已发表（不能挂未发表的文章）
- 文章不能违规、不能太短

## 公众号文章定稿来源

所有 7 篇文章已写在 `docs/wechat-7day-content.md`（公众号 7 天发布文案）。Manus 不需要重写，只需在视频号发布前确认公众号侧已按这份文案发表。

发表顺序：
- T-1 天 14:00 前：公众号发表当天对应那篇
- T+0 当天发布时间：视频号发表，挂当天那篇公众号文章

## 注意事项

1. **视频号 D1 → 公众号 D1**，但视频号 D2 → 公众号 **D3**（不是 D2）— **不是按公众号顺序挂**，是按主题对应挂。务必看上表。

2. 视频号底部扩展链接只能挂 1 篇公众号文章。**不要试图在评论区再放公众号链接**（视频号反垃圾会限流）。

3. 公众号文章发表后**不要删除**（删除后视频号扩展链接会变成"文章已删除"，体验极差）。

4. 公众号文章末尾的 "立即体验 enclaveai.top" + "加群回复'进群'" 已在原文里，无需修改。

5. 如果 7 篇公众号文章已经发过（往期推文），扩展链接也能挂往期推文，**无需重新发**。

## 数据回看

每条视频发布 24h 后，去视频号助手看：
- "扩展链接点击数" / 视频播放数 = 公众号文章导流率
- 健康值 ≥ 3%（视频号专属指标）

如果某条视频点击率明显低（<1%），说明文章标题/主题与视频钩子断了 — 可以替换为另一篇更相关的（不必非按 7 天对照表，可灵活调整）。
