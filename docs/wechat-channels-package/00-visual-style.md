# 视觉规范（视频号专属版）

> **本文档与抖音包 `docs/douyin-package/00-visual-style.md` 99% 相同**。Manus 应先完整阅读抖音版规范，本文件只列出**视频号特化差异点**。

---

## 与抖音版的差异点（仅看这部分即可）

### 1. UI 安全区（核心差异）

抖音 UI 占位与视频号 UI 占位不同。视频号底部多了"扩展链接"卡片占位（挂公众号文章用），所以底部安全区**比抖音多 40px**：

```
                    抖音           视频号
顶部状态栏占位       0-200px        0-180px      （视频号顶部状态栏更短）
底部 UI 占位         0-280px        0-320px      （视频号底部 + 扩展链接需要更多空间）
右侧操作栏占位       0-180px        0-160px
左下文案占位         0-200×0-400   0-180×0-360
```

**字幕区调整**：原抖音版字幕 Y=1200-1480 → 视频号版 **Y=1100-1480**（上移 100px，避开底部"扩展链接"卡片）。

### 2. 时长

- 抖音：28-45s
- **视频号：60-90s**（每条 shot-list 顶部明确）

### 3. 配音节奏

- 抖音：270 字/分（快钩子）
- **视频号：240-260 字/分**（慢一点，给情感留呼吸）

### 4. 钩子节奏

- 抖音：3 秒内必须建立
- **视频号：5-7 秒可缓一些**（首句字幕 fade-in 1-1.5s 都是允许的）

### 5. BGM 选择

- 抖音：抖音商用曲库
- **视频号：使用 CC0 / Pixabay Music / Bensound**（视频号商用曲库较小，CC0 库更安全）
- 推荐：`Pixabay - Calm Piano`、`Bensound - Sweet`、`Pixabay - Lo-fi Study`
- **片尾 1 秒 BGM 切断**（不要淡出）— 与抖音版一致

### 6. 片尾

抖音版片尾 logo 1-1.5s。**视频号版片尾 logo 1.5-2s**（多 0.5s 让观众有时间点击底部"扩展链接"看公众号文章）。

## 完全复用抖音版的部分（不再赘述，请阅读 `docs/douyin-package/00-visual-style.md`）

- ✅ 画面规格：1080×1920 / 30fps / H.264 / AAC 192kbps
- ✅ 品牌色：`#f97316` 主橙 / `#0a0a0a` 深底 / `#fef7ed` 米色
- ✅ 字幕样式：思源黑体 Bold / 70px 白字 / 2px 黑描边 / 阴影 / 行距 1.3 / 单行 14 中文字
- ✅ 强调字：88px / `#f97316` / 1 帧抖动
- ✅ 转场限制：硬切默认 / 0.2s 闪白 ≤2 次/条 / 0.3s 黑场 ≤1 次/条 / Cross dissolve 禁用
- ✅ 录屏画面包装：12px 圆角 + 2px `#262626` 描边 + 模糊扩展背景
- ✅ 字幕进入节奏：先于配音 200ms 出现，配音结束后停留 ≥800ms
- ✅ BGM 处理：人声 0dB / BGM -22 至 -18dB，重点台词 -26dB
- ✅ 文件命名规范：`output/day-N/d{N}_video.mp4` etc.

## 视频号"扩展链接"卡片预览

视频发布时附挂公众号文章后，视频号底部会出现这样的卡片：

```
┌─────────────────────────────────────┐
│  [视频内容区]                        │
│  （字幕区移到 Y=1100-1480）          │
├─────────────────────────────────────┤
│ 📄 公众号文章标题（24px 白底卡片）   │
│    公众号名称 · 阅读 X · ¥          │
└─────────────────────────────────────┘
```

字幕**绝对不能盖到这张卡片**。建议保留至少 60px 间隙。

## 视频号 UI 安全区可视化

```
顶部 0-180px       → 视频号状态栏 + 顶部下拉
底部 0-320px       → 视频号底部 tab + 扩展链接卡片
右侧 0-160px       → 视频号右侧操作栏（点赞/评论/分享/直播按钮）
左下 0-180×0-360  → 视频号文案区
```

字幕、录屏画面、品牌大字、logo 都必须避开上述区域。
