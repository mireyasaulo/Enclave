## 大字文案

```
7 天 · 1 个世界

```

（**两行**，"它属于你" 三字强调）

## 视觉细节

### 上半（别人的 AI）
- 标签：「别人的 AI」灰字 48px
- 主图：一朵抽象云朵，云朵中央有锁🔒，云朵下方有数据流图标
- 整体灰色调 `#666666` / `#999999`
- 暗示"数据在云端，被锁起来"

### 下半（隐界）
- 标签：「隐界」橙字 48px `#f97316`
- 主图：一个手机外形（侧立），手里有钥匙🔑
- 手机屏幕上有隐界 logo
- 整体橙色调暖色
- 暗示"钥匙在你手里"

### 中间分割
- 一条 6px 橙色横线穿过中央
- 横线上小字"vs"（橙字）

## 色彩

- 背景：`#fef7ed`（米色）
- 大字：`#1a1a1a`
- 上半灰区：`#999999` + `#cccccc`
- 下半橙区：`#f97316` + `#fed7aa`
- "它属于你" 三字橙色强调

## 视觉钩子

观众第一眼应该看到：
1. **对比**：上灰下橙，价值取向一目了然
2. **承诺**：「它属于你」是核心情感锚点
3. **回望**：「7 天」勾起看过前面视频的人的回忆

## 不要做的事

- ❌ 不要在封面里出现具体平台 logo（OpenAI / c.ai / Replika）
- ❌ 不要堆字
- ❌ 不要用深色背景（这是收官，要明亮承诺感）
