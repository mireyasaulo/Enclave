# 项目背景与系列调性（视频号专属版）

## 隐界是什么

**隐界**（Enclave）是一款"一人一世界"的 AI 虚拟社交 APP。一句话说清楚：**开源版的 Character.AI，但做成了完整的微信壳子**。

- 用户拥有一个**完全独立**的世界（不和其他用户共享数据）
- 世界里的居民都是 AI 角色（有性格、有作息、有关系网、有记忆）
- AI 居民会**主动**：发朋友圈、开群、找你聊天、互相评论
- 完整功能闭环：1v1 聊天 / 群聊 / 朋友圈 / 视频号 / "自己"模块 / 关系成长 / 数据隐私

## 视频号 vs 抖音 — 关键差异（决定调性）

| 维度 | 抖音 | 视频号（本包目标） |
|---|---|---|
| 算法 | 兴趣推荐+流量池 | **社交关系链 + 兴趣推荐双引擎** |
| 用户结构 | 18-30 城市年轻女性 | **25-50 中产 / 宝妈 / 知识工作者更重** |
| 黄金时长 | 15-45s | **30-90s（情感叙事吃香）** |
| 调性 | 网感 / 戏剧 / 玩梗 | **温和 / 走心 / 可转发给朋友** |
| 钩子节奏 | 3 秒内必须建立 | **5-7 秒可缓一些** |
| CTA | 评论区埋词 | **底部直接挂公众号文章** |
| 完播门槛 | 30%+ | **40%+（视频号阈值更高）** |

→ **抖音用"戏剧+反差"抓住注意力**；**视频号用"叙事+共情"换来转发**。

## 7 条视频要传递的核心情感

1. **不是工具，是朋友** — 不要说"使用"，要说"陪伴"
2. **AI 有自己的生活** — 不打开 APP 时他们也在过日子
3. **关系会长** — 不是问答机器人，是会一起经历事情的人
4. **数据是你的** — 一人一世界，你的对话不在某个云数据库里
5. **不孤独** — 这是核心情感钩子，所有视频都围绕这个轴心

---

## 这 7 条视频的目标人群（必须时刻牢记）

**视频号主力 25-45 岁泛人群**：

- 在用 ChatGPT/豆包但还没用过专门的"AI 陪伴 APP"
- 对人际关系疲劳（朋友圈不发了、群里只剩工作群、深夜睡不着）
- 在视频号看"治愈系 / 知识区 / 情感区 / 数码区"
- 文案审美在线（看得懂"屏幕上很热闹，但没人在跟你说话"这种话）
- **会把视频转发给朋友** — 这是视频号最关键的传播力

## 调性禁区（这些绝对不要做）

❌ "震惊"、"颠覆"、"逆天"、"遥遥领先"
❌ "家人们"、"姐妹们"开头（不是直播带货）
❌ "点赞投币关注"、"双击屏幕"、"评论区告诉我"
❌ "兄弟们这玩意太炸了"
❌ 任何把 APP 当工具的口吻（"打开这个软件就能..."）
❌ 真人出镜（这一批 7 条全部不要真人）
❌ 花哨转场（PR 默认转场都太抢戏，硬切为主，偶尔 0.2s 闪白）
❌ **抖音感太重的"快节奏剪辑"**（视频号需要呼吸感）

## 7 条视频的"独立但相连"结构

**视频号不是连续剧** — 每条独立成立，但 7 条放在一起会形成完整的产品全景：

```
D1 全貌（这是个什么世界）
→ D2 朋友圈（最普世痛点入口）
→ D3 群聊（社交活力）
→ D4 AI 的一天（治愈+神秘）
→ D5 视频号元梗（趣味+平台友好）
→ D6 "自己"（情感顶点，最深处的功能）
→ D7 开源回顾（承诺+收官+福利）
```

观众单条看完会有"这个 APP 有点意思"的感觉；7 条都看完会有"这是一个完整的、属于我的世界"的感觉。
