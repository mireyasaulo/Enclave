## 3. 素材使用约定

每个镜头在 `shot-list.md` 都会标注素材来源，三类：

- **🖼️ EXISTING** — 直接使用本包内的现成素材，**不能用 AI 重画**（路径会精确给出）
- **🎨 GENERATE** — 镜头里给出提示词，由 Manus 用图像生成模型产出（**必须符合品牌色 `#f97316`**）
- **📹 SCREEN-CAP** — 屏幕录制段。**本包已为每条视频写好"应在 APP 内出现的对话/朋友圈数据"**（见 `day-N/data/`），Manus 应基于这些数据**用 Figma / Sketch / 设计工具复刻聊天 UI 样式**，**不要凭空捏造内容**

⚠️ **核心原则**：所有出现在视频里的聊天内容、朋友圈内容、群聊内容，**必须严格等于 `data/data.json` 里给定的文本**。任何一字之差都不行——这些内容是项目方在产品里真实生成的，是视频可信度的基础。

⚠️ **UI 复刻参考**：
- `shared-assets/product-screenshots/` 下有 6 张 APP 真实 UI 截图，**复刻聊天 UI 时严格参照其布局、色彩、间距、气泡形状**
- `shared-assets/brand/colors.json` 是品牌色完整定义
- `shared-assets/brand/logo.svg` 是 logo 矢量源
- `shared-assets/avatars/` 是每个角色的真实头像（SVG/PNG），**不要 AI 重绘**

## 4. 旁白 / 配音规范（视频号特化）

- 旁白稿在 `day-N/script.md`，已分段标注时间码
- **音色**：年轻女性，**24-28 岁声线**，温和/治愈/有重量的混合质感
- **语速**：**视频号 240-260 字/分钟**（比抖音慢一点，让叙事有呼吸感）
- **情感曲线**：钩子段平稳 → 中段稍带情绪 → 收尾克制（不要嗨/不要喊/不要"姐妹们"开头）
- **不要说**：「家人们」「姐妹们」「点赞收藏关注」「评论区告诉我」（评论区埋钩子，不在视频里喊）

推荐 TTS 音色（7 条必须用同一音色）：
- minimax `Chinese (Mandarin)_Lovely_Girl` ✅ 首选
- 火山 `zh_female_qiniao`
- 阿里云 cosyvoice `晓晨`

## 5. 风险防控（视频号特化）

- **可以挂公众号链接**（视频号原生扩展链接能力，与抖音不同）
- **不出现「ChatGPT」「GPT」「AI 女友」**（敏感词限流） — 一律用「AI 朋友」「他」「她」
- **不出现其他平台 logo**（抖音、小红书、B 站、X、TikTok），仿真截图打码处理
- **不念出 `github.com/...`**（用「开源」、「可以自己部署」绕开）
- **不出现真实手机号/邮箱/IP**（如截图里出现要打码）
- **BGM 必须用视频号商用曲库**（视频号侵权下架同样严格），推荐 CC0 / Pixabay Music

## 6. 与抖音包 / 公众号包的关系

本仓库已有：

- `docs/douyin-package/` — 抖音 7 天包（30-32s 短钩子，主角"失眠女青年"暗叙事线）
- `docs/wechat-7day-content.md` — 公众号 7 天图文（每篇深度长文）

视频号包**调性、剧本、时长全不同**，但视觉规范（字幕样式/品牌色/录屏包装）和 shared-assets 完全复用抖音包（已通过 symlink 共享）。

视频号视频负责"情感钩子"，公众号文章负责"深度种草"，形成**视频号 → 公众号 → 官网体验** 三段闭环。每条视频底部挂的公众号文章对照表见 `00-public-account-link-map.md`。
