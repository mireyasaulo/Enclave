## 30 秒概览

- **任务**：生产 7 条微信视频号短视频（每条 60-90s，竖屏 1080×1920）
- **形式**：APP 录屏 + 中文女声旁白 + 硬烧字幕（不要真人出镜）

## 生产 1 条视频的完整流程

### Step 1：读当天的 5 份文档

`day-N/` 下：
- `script.md` — 旁白逐字稿（带时间码，直接喂 TTS）
- `shot-list.md` — 镜头清单（每个镜头：素材路径 / 字幕 / 音效）
- `cover-spec.md` — 封面规格
- `publish-meta.md` — 发布文案 + 标签 + 评论区第一条
- `data/data.json` — APP 内要出现的真实对话/朋友圈/群聊文本

### Step 2：准备 4 类素材

| 类型 | 来源 | 怎么处理 |
|---|---|---|
| 🖼️ EXISTING | `shared-assets/` 内已有 | 直接用，不要 AI 重画 |
| 🎨 GENERATE | shot-list 标注为 GENERATE | 用图生图模型按 `GENERATE-ASSET-PROMPTS.md` 里的精确 prompt 生成 |
| 📹 SCREEN-CAP | `data/data.json` 里给的真实文本 | 用 Figma/Sketch 复刻 APP UI 样式，**文字必须字字对应 data.json**，不要捏造 |
| 🎵 BGM | CC0 库（Pixabay / Bensound） | 见 shot-list 里的推荐曲目 |

### Step 3：TTS 配音

- **音色统一**：7 条必须同一个 TTS 配置
- **首选**：minimax `Chinese (Mandarin)_Lovely_Girl`
- **备选**：火山 `zh_female_qiniao` / 阿里云 cosyvoice `晓晨`
- **语速**：视频号 240-260 字/分（比抖音慢）
- **风格**：24-28 岁女声，温和有重量，不要播音腔，不要"姐妹们"开头

### Step 4：剪辑合成

按 shot-list 顺序拼镜头：
- 字幕硬烧（不要外挂 SRT），样式见 `00-visual-style.md` 第 3 节
- 字幕区 **Y=1100-1480**（视频号特化，避开底部扩展链接卡片）
- 转场默认硬切，每条最多 2 次 0.2s 闪白 + 1 次 0.3s 黑场
- 录屏画面 12px 圆角 + 2px `#262626` 描边 + 模糊背景扩展

### Step 5：自检 13 条

按 `00-master-checklist.md` 逐项确认。**任何一项 ❌ 不交付。**

### Step 6：产出归档

```
output/day-N/
├── d{N}_video.mp4           主视频（1080×1920 / 30fps / H.264）
├── d{N}_cover.jpg           封面（1080×1440）
├── d{N}_publish.txt         发布文案 + 标签（从 publish-meta.md 复制）
├── d{N}_pinned.txt          评论区第一条（从 publish-meta.md 复制）
└── d{N}_public-link.txt     视频号底部挂的公众号文章标题
```

## 红线（一旦违反必返工）

- ❌ 不出现「ChatGPT」「GPT」「AI 女友」字样
- ❌ 不出现其他平台 logo 清晰版（抖音/小红书/微信/B 站要打码至少 40%）
- ❌ 不出现 `github.com` 或任何明文 URL
- ❌ 字幕、录屏画面不能侵入视频号 UI 安全区（顶 180 / 底 320 / 右 160）
- ❌ 7 条不能用不同 TTS 音色（必须统一）
- ❌ APP 内对话/朋友圈/群聊文本与 data.json 不一致
