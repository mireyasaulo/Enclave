# 待生成素材 · 精确提示词清单

> 本文档列出 D1-D7 所有 🎨 GENERATE / 🔲 待生成素材的精确提示词。
> Manus 用图生图模型（如 Midjourney / SDXL / Imagen / Flux）或设计工具（Figma / Photoshop）按这些 prompt 生成。
> **共通规范**：所有图片输出为 PNG（1080×1920 或镜头要求的尺寸），符合品牌色 `#f97316`，**严禁出现其他真实平台 logo 清晰版**。

---

## 通用风格基线（所有图通用）

```
品牌色：橙色 #f97316（主色）+ 米色 #fef7ed（暖底）+ 深灰 #1a1a1a（文字）
风格：极简 / 治愈系 / 视频号审美（不要小红书那种过度饱和，不要抖音那种闪光特效）
材质：纸质感 + 微噪点（非塑料感）
不要：渐变彩虹 / 撞色 / 卡通过度可爱 / 真人脸 / 任何品牌 logo 清晰版
```

## 通用：仿真截图的 9 条注意事项

1. **永远不要清晰露出其他平台 logo**（抖音、微信、微博、小红书、ChatGPT、Character AI、Replika、OpenAI）— 必须模糊 ≥ 30% 或者用通用图标替代
2. **不要真人脸**（用默认头像、抽象剪影、动物代替）
3. **不要 github.com 明文 URL**（必须打码）
4. **配色统一**：品牌色 `#f97316` 主橙 / `#fef7ed` 米色 / `#1a1a1a` 深灰
5. **不要渐变彩虹字**（视觉规范禁止）
6. **不要堆 emoji**（最多每张图 1-2 个）
7. **手写体只用在 D2-B 采访答案**（其他地方用思源黑体 Bold）
8. **所有 PNG 输出 1080×1920** 或 shot-list 注明的尺寸
9. **所有素材必须 CC0 / 商用许可**（不可以下载侵权图）

## 提示词使用建议

每个 prompt 我已经写得相对完整，但你（Manus）可以：
- 根据图生图模型的特性微调
- 加入风格指令（如 "minimalist illustration" / "flat design"）
- 第一次出图不满意请按 prompt 描述的"核心元素 + 配色 + 风格基线"重做
- 不要为了出图效果牺牲红线（侵权风险）

如果某个 prompt 出图后你判断不符合视频号调性（过于戏剧/抖音感），按 "00-series-context.md" 调性范本重新出。
