## 视觉细节

### 群聊截图（中央主体）
- 顶部：群名「测试群」+ 6 拼头像 + 在线人数"5/6"（暗示 1 个退群了）
- 4 条对话气泡（按 data.json 老张/林晨对话内容）：
  1. 老张："不点变态辣的都是叛徒"（红橙色气泡 + 沸腾 emoji）
  2. 林晨："上次让我吃，第二天我胃疼"（灰气泡 + 委屈 emoji）
  3. 老张："娇气"（红橙短气泡）
  4. 系统提示条："林晨已退出群聊"（灰底红字）
- 截图整体加 12px 圆角 + 2px 描边 `#262626`

## 色彩

- 背景：`#fef7ed`（米色）
- 大字：`#1a1a1a`
- 群聊气泡：白底（其他角色）+ 橙色 `#f97316`（用户）
- 系统提示红字：`#dc2626`

## 视觉钩子

观众第一眼应该看到：
1. **戏剧**：「吵到第 17 楼」数字醒目
2. **冲突**：截图里能看到红气泡 vs 灰气泡的对立
3. **悬念**：「退群了」红字立刻让人想点开
