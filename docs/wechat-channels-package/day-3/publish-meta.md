## 标签策略

| 标签 | 池子大小 | 视频号功能 |
|------|---------|---------|
| `#群聊` | 大 | 主题精准 |
| `#ai陪伴` | 中 | 精准产品池 |
| `#沙雕日常` | 大 | 戏剧标签，引流广 |
| `#谁懂啊` | 大（热门） | 蹭热门 |
| `#治愈系` | 大 | 主流量 |

**5 个标签**（视频号建议 5-8，戏剧类内容标签略少更聚焦）

## 评论区第一条（小号置顶）

```
笑死，这群比我工作群还热闹
```

**目的**：
- 让观众产生"这是真实评论"的错觉（"笑死"是小红书/视频号高频口语）
- 引导其他人评论"我也想看看"
- 制造转发欲望
