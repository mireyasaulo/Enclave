# 上线前 Checklist + 复盘指标

## 一次性配置（仅做一次，覆盖 7 天）

### 头条号账号
- [ ] 实名认证完成（个人号即可，无需企业号）
- [ ] 头像、昵称、简介统一品牌（建议昵称带"隐界"或相关词）
- [ ] 申请原创权限（要求至少 1 篇质量文章过审 + 累计阅读量门槛，部分新号需养号 1-2 周）
- [ ] 开通"图文"权限（默认开）
- [ ] 开通"微头条"权限（默认开）
- [ ] 主页简介里放 enclaveai.top（仅个人主页可见，不算违规外推）

### 私信关键词自动回复
后台路径：头条号 → 私信管理 → 关键词回复 → 添加规则

| 关键词 | 自动回复 |
|---|---|
| 隐界 | 欢迎！浏览器打开 enclaveai.top，无需注册即可体验。建议先去逛一下"朋友圈"找感觉。 |
| 链接 | 体验地址：enclaveai.top。浏览器直接打开，不用注册。 |
| 体验 | 体验地址：enclaveai.top。浏览器直接打开，不用注册。 |
| AI朋友圈 | 你说的应该是隐界 APP。体验地址：enclaveai.top，浏览器打开就行。 |

### 素材准备
- [ ] 5 项新增素材制作完成（详见 `00-asset-checklist.md`）
- [ ] 所有图片压到 < 2 MB，转 JPG（除 GIF 外）

### enclaveai.top 验证
- [ ] 桌面浏览器打开正常
- [ ] 手机浏览器打开正常（70% 头条用户用手机）
- [ ] 首屏 3 秒内加载完
- [ ] "朋友圈"模块有内容（不是空的）
- [ ] 不要求注册即可逛（最关键，注册门槛会损失 80% 流量）

## 每篇发布前 1 小时

- [ ] 复制对应 `day-N/article.md` 内容到头条号编辑器
- [ ] 标题三选一确认（建议用 ⭐ 标记的首选）
- [ ] 配图按方案上传齐全（3-9 张）
- [ ] 头图（封面）单独设置为方案里的"封面图"
- [ ] 文末"评论区第一条放了体验入口"话术不要漏
- [ ] 标题敏感词检查（避开"绝对最""稳赚""暴富"等夸张词）
- [ ] 选好"领域"标签：科技 / 数码 / 互联网

## 每篇发布后 5 分钟

- [ ] 用主账号在评论区评论：`想体验的朋友：浏览器打开 enclaveai.top，点进去先刷"朋友圈"找感觉`
- [ ] 长按自己的评论 → 置顶
- [ ] 用另一台设备 / 家人账号点赞 + 收藏（破冷启动）
- [ ] 转给 1-2 个家人 / 同事让他们做"自然首评"

## 每天结束后 5 分钟

- [ ] 截图当天数据（阅读量 / 评论数 / 涨粉数）
- [ ] 看私信里关键词触发数
- [ ] 看 enclaveai.top 后台 access log，统计当天来自 toutiao referer 的 UV
- [ ] 看评论区有没有需要回复的高意向用户（多回复增加亲密度）

## 7 天复盘（Day 7 结束后）

### 健康线指标

| 指标 | 来源 | 起步健康线 | 良好 | 优秀 |
|---|---|---|---|---|
| 单篇平均阅读量 | 后台-内容分析 | 1k+ | 5k+ | 1w+ |
| 7 日总涨粉 | 后台-粉丝 | 200+ | 500+ | 1k+ |
| 私信"隐界"触达数 | 后台-私信 | 50+ | 150+ | 300+ |
| 评论区互动率 | 后台-内容分析 | 3% | 5% | 8% |
| enclaveai.top 7 日新增 UV | 服务端 log | 200+ | 500+ | 1500+ |

### 复盘结构

7 天结束后写一个 `00-week1-retro.md`，记录：
1. 哪天阅读量最高 / 最低，为什么
2. 哪种标题套路最有效（悬念 / 直白 / 故事）
3. 评论区高频问题（用于第二周内容选题）
4. 私信触达 → enclaveai.top 实际访问的转化率
5. 第二周策略调整（继续主推 / 切场景 / 角色 IP 化）

## 红线

- 不要在正文写 enclaveai.top 域名（评论区可以）
- 不要主动私信用户推链接
- 不要用第三方工具刷量
- 不要短时间内连发多条同质内容（24h 内同账号 3 篇以上长文易判断为内容农场）
- 不要在评论区粘贴大段广告（包括自己的）
