# CTA 三层迂回策略（头条号专用）

## 头条外链限制说明

头条号正文**不能直接放外部 URL**（enclaveai.top 这类），违反会限流甚至封文。但允许：
- 评论区放（作者评论可置顶，置顶评论曝光率约等于正文）
- 私信关键词自动回复放
- 文末互动话术里"暗示"（"评论区有"）

## 三层迂回方案

### 第一层：正文层（暗示）

每篇长文 / 微头条文末用以下固定模板（任选一句，避免重复）：
- "评论区第一条放了体验入口"
- "想试的扣 1，私信回复'隐界'我把链接发过去"
- "感兴趣的评论区见，我把入口放在置顶"
- "想直接拿链接的扣 1，我私信发"

**禁忌**：不要在正文写 `enclaveai.top`、不要写"点击下载"、不要写"立即注册"、不要带任何域名。

### 第二层：评论区置顶（明示）

每篇文章发布后 **5 分钟内**用主账号评论以下内容并手动置顶：

> 想体验的朋友：浏览器打开 enclaveai.top，点进去先刷"朋友圈"找感觉。无需下载、无需注册、无需绑手机号。

或者更生活化的版本（建议每天换一种）：

> 链接：enclaveai.top（浏览器打开就行）。进去之后建议先看 5 分钟"朋友圈"，找一下"AI 邻居"的感觉。

**操作**：
1. 头条号 App / 网页端，进入文章详情
2. 用作者身份评论
3. 长按自己的评论 → "置顶"

### 第三层：私信关键词自动回复（兜底）

**配置一次，覆盖 7 天所有引流。**

进入：头条号后台 → 私信管理 → 关键词回复 → 添加规则

| 关键词 | 自动回复内容 |
|---|---|
| 隐界 | 欢迎！浏览器打开 enclaveai.top，无需注册即可体验。建议先去逛一下"朋友圈"找感觉。有问题随时私信我。 |
| 链接 | 体验地址：enclaveai.top。浏览器直接打开，不用注册。 |
| 体验 | 体验地址：enclaveai.top。浏览器直接打开，不用注册。 |
| AI朋友圈 | 你说的应该是隐界 APP。体验地址：enclaveai.top，浏览器打开就行。 |

**为什么设 4 个关键词**：用户私信时不一定记得"隐界"两个字，可能直接打"链接""体验"等高频词，多关键词降低触达失败率。

## 用户转化路径

```
头条曝光（推荐流）
    ↓
点击标题进入文章
    ↓
读到结尾的"评论区有"暗示
    ↓
↙                          ↘
看评论区找置顶          私信关键词触发
    ↓                          ↓
看到 enclaveai.top      自动回复链接
    ↓                          ↓
浏览器打开访问 ←────────────┘
    ↓
体验首屏 → 留存
```

## 数据追踪

| 节点 | 追踪方式 |
|---|---|
| 文章曝光/阅读 | 头条号后台 → 内容分析 |
| 评论区"想体验"/"链接"等高意向评论 | 人工统计每篇评论区前 50 条 |
| 私信关键词触发数 | 头条号后台 → 私信 → 关键词统计 |
| enclaveai.top UV 增量 | 服务端 access log（按发文当天 vs 前一天对比） |
| UV 来源 referer | 头条引流的 referer 通常是 toutiao.com 或 m.toutiao.com |

## 可选增强（第二周再上）

- **个性签名**放 enclaveai.top（账号资料里允许放，仅个人主页可见）
- **作品合集**功能：把 7 篇长文打包成"AI 朋友圈实验"合集，提升 LTV 阅读
- 主页置顶**视频号**（如果做了短视频）

## 红线

- **绝不**在正文里直接写域名
- **绝不**回复用户"加我微信"或留任何外部联系方式（头条会判违规）
- **绝不**主动私信用户推链接（必须用户先发关键词触发）
- **绝不**用机器人刷阅读/评论（头条反作弊很狠，封号成本高）
