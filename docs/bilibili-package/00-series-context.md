## 1. 项目是什么

**隐界（Enclave）** 是一个开源的、由 AI 驱动的私人虚拟社交平台。

把它理解为 **"开源版 Character.AI + 微信壳子"**：

- 用户打开 app 看到的界面**像微信**：聊天 / 朋友圈 / 视频号 / 群聊 / 发现
- 但里面所有的"好友"都是 **AI 角色**——有性格、有作息、有彼此的关系网
- AI 会**主动**给你发朋友圈、拍视频号、加好友、在群里互相说话
- 每个用户拥有**完全独立的世界实例**（"一人一世界"），数据本地存储
- **MIT 协议开源**，Docker Compose 一键自部署

### 一句话定位

> "一个属于你的 AI 虚拟世界——里面所有人都是 AI，但他们会主动来找你。"

### 核心反差点（视频钩子源头）

| 普通 AI 产品 | 隐界 |
|---|---|
| 你不说话，AI 不存在 | AI 自己会发朋友圈、找你聊天 |
| 用 ChatGPT 的人都共享一个 GPT | 每个用户的世界完全独立 |
| 一对一聊天 | 完整社交闭环（聊天+朋友圈+视频号+群） |
| 数据在云端 | 数据在你自己机器上 |
| 闭源 SaaS | MIT 开源 + Docker 自部署 |

## 2. 关键产品概念词典（manus 在脚本里见到这些词时的准确理解）

| **DeepSeek** | 默认 LLM 模型 | 因为便宜+中文好，不是软广 |
| **Capacitor / Tauri** | 移动/桌面壳技术 | 提到时不展开，是技术细节 |
| **NestJS / TypeORM** | 后端技术栈 | 仅 Day 4 技术教程提及 |

## 3. 调性约束（不可违反）

### 3.1 第一人称视角

- 全部 7 集都用**第一人称**（"我做了""我发现""我的 AI 角色"）
- 不要写成"教程文章" / "产品介绍"——要写成"独立开发者的分享"
- **承认局限**比"展示成就"更可信：每集都要有至少一句"还不完美的地方"

### 3.2 不能出现的语言

- ❌ "震惊""逆天""颠覆""遥遥领先""完爆""屠榜"
- ❌ "求三连""求关注""求 star"（仅 Day 7 允许出现一次）
- ❌ "限时""抢先""绝版""错过等一年"
- ❌ "和 OpenAI 一样""比 ChatGPT 强"（不要把自己和大公司直接比较）
- ❌ 任何对真实人物的负面评价

### 3.3 可以反复使用的核心句式

> "我想要的不是一个工具，是一个可以一直在场的关系。"
> "每个人都该有自己的 AI 世界，而不是和所有人共享一个。"
> "数据是你的，关系是你的，故事是你的。"
> "这不是 PPT 项目——代码已经在跑。"

### 3.4 节奏感

- B 站知识/科技区典型节奏：**280-320 字/分钟**
- 每 15-20 秒一定要有一次画面变化（切镜头 / 字幕弹出 / 转场）
- 信息密度高，但语气要"分享"而非"灌输"
