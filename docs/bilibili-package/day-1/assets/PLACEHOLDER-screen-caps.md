## 录屏要点

- **不要使用真实人物头像**——用项目默认角色头像即可
- **不要泄露真实手机号 / 邮箱 / IP**——切到 app 之前清空桌面通知
- **时间戳必须真实**——不要后期 PS 改时间，会被识破
- **如有 LLM 回复慢的情况**，正常等待，剪辑时压缩即可（保留"等待感"反而真实）
