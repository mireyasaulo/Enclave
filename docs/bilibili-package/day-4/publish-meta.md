## 置顶评论

```
完整部署命令清单：

1. git clone https://github.com/yuanzui0728/enclave.git
2. cd enclave
3. cp api/.env.example api/.env
4. 编辑 .env 填入 DEEPSEEK_API_KEY 和随机 ADMIN_SECRET
5. docker compose up -d
6. 浏览器打开 http://localhost

部署遇到 better-sqlite3 编译报错的：Dockerfile 里 alpine 镜像需要 python3+make+g++（已经默认装好），如果你改了基础镜像记得手动加。

Issue 区可以问：github.com/yuanzui0728/enclave/issues
```
