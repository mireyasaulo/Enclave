## 第 1 段 · 你需要准备什么 [1:00]

> 三样东西：
> 第一——**Docker**。
> 这是个开源的容器工具，官网搜 Docker Desktop，Windows / Mac / Linux 都有。
> 装好后能在终端运行 `docker --version`，就 OK 了。
>
> 第二——**一把 DeepSeek API Key**。
> 去 platform.deepseek.com 注册，赠送的免费额度足够你跑很多天。
> 不想用 DeepSeek 也可以——OpenAI、通义、Groq、Ollama 都支持，待会儿说怎么换。
>
> 第三——**一台 2C2G 以上的机器**。
> 实测 2 核 2G 内存的小 VPS 完全能跑。我自己就在一台百块的小机器上跑。

---

## 第 2 段 · 步骤 1 - clone 仓库 [2:00]

> 打开你的终端，开始三条命令的第一条：
> ```
> git clone https://github.com/yuanzui0728/enclave.git
> cd enclave
> ```
> 项目克隆完。
>
> 接下来配置环境变量：
> ```
> cp api/.env.example api/.env
> ```
> 这会复制一份配置模板。
> 用任何文本编辑器打开 `api/.env`，找到两个必须填的字段——
> 一个叫 `DEEPSEEK_API_KEY`，把你刚才从 DeepSeek 平台拿到的 key 粘进去。
> 一个叫 `ADMIN_SECRET`，随便填个长字符串就行——这是你这个世界的"主人密码"。
>
> 其他字段先不用动，全部默认。

---

## 第 3 段 · 步骤 2 - 起服务 [3:30]

> 一条命令，启动整个项目：
> ```
> docker compose up -d
> ```
> -d 是后台运行的意思。
> 第一次启动会下载镜像，大概一两分钟。
>
> 我们看看是不是真的起来了：
> ```
> docker compose ps
> ```
> 应该能看到四个容器全部 Up：api、app、admin、nginx。
>
> 再看看资源占用：
> ```
> docker stats --no-stream
> ```
> 我的机器上，CPU 闲时几乎是 0%，内存大概 350MB 左右。
> 2 核 2G 的机器完全扛得住。

## 第 4 段 · 步骤 3 - 第一次使用 [5:30]

> 浏览器打开 `http://localhost`。
> 第一次会跑一个**单例迁移**——把你设为这个世界的"主人"。
> 完事之后，你会经历一段**5 幕入场动画**——5 个 AI 角色依次亮相，是为了让你第一次感受这个"世界"。
>
> 看完入场动画，你会进入主界面。
> 来试一下"摇一摇"——这是一个发现新角色的入口：
> 摇一摇，会随机出现一个 AI 角色，附带一段他**主动生成**的加好友申请文案。
> 比如这个角色说："看你最近在做产品，想认识一下，我做过几年游戏设计。"
> 同意加好友，他就进入你的世界了。
>
> 给他发一条消息试试——这是你和他的第一段对话。

## 第 5 段 · 常见问题 [6:30]

> 几个最常被问的问题——
>
> **Q：能不能换成 OpenAI、Ollama？**
> 能。隐界默认走 OpenAI 兼容协议，改 `.env` 里的 `OPENAI_BASE_URL` 和 `OPENAI_API_KEY`，一行的事。
> Ollama 本地跑也行——把 baseURL 改成 `http://host.docker.internal:11434/v1` 就接通了。
>
> **Q：我的数据会不会上传？**
> 不会。所有数据都在你机器上 `./data/cloud-platform.sqlite` 这一个文件里。
> 你想备份，把这文件 cp 一下；你想删，rm 一下就消失了。
>
> **Q：能不能远程访问？**
> 能。在前面套一层 frp / cloudflare tunnel / 花生壳都行。
> 我自己就用花生壳的 vicp.fun 暴露在公网，给陌生人试玩。
