## 文字层

### 主标题
- 文字：「7 天回头看」
- 字号：180px
- 颜色：#FFFFFF
- 字体：思源宋体 Black（继承 Day 5 的宋体感，呼应"收尾"严肃感）

### 副标题（数据占位符）
- 内容：「+{{STAR_COUNT}} star · +{{FAN_COUNT}} 粉丝」
- 字号：60px
- 颜色：#4ADE80（绿，正向数据感）
- 字体：JetBrains Mono Bold（数字感）
- **⚠️ 占位符**：manus 可生成 1 张默认（含占位符可见）+ 1 张备选（不写数字），UP 主当天替换或选用

## 图像层

### 背景
- 主色：白底渐变到浅灰 #F3F4F6（区别于前 6 集的深色！）
- 顶部加淡蓝色调

⚠️ 收官集是 7 集里**唯一的浅色背景**封面——视觉上明确"系列结束、清算时刻"。

### 右上 Star History 曲线
- 🎨 GENERATE 提示词：
  ```
  Stylized line chart showing GitHub star history over 7 days,
  line going up dramatically. Clean modern data-viz style, blue gradient
  under the line, transparent background
  ```
- 位置：右上 1/3 区域

### 底部 7 个集子缩略图横排
- 直接抽取每集 cover 的左侧 200×200 区域作为小图
- 横排：D1 / D2 / D3 / D4 / D5 / D6 / D7
- 每个小图加圆角 + 8px 间距
- 在 D7 位置可以加一个"完结"角标
