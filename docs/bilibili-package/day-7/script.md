## 第 3 段 · v0.2 路线图 [3:00]

> 接下来 4-6 周要做的事——
>
> **第一：长期记忆向量召回。**
> 这是隐界当前最大的短板——AI 记不住你们前几周聊过的事。
> 准备改成多策略召回（时间衰减 + 角色过滤 + 主题聚类）。
>
> **第二：多语言模型路由。**
> 中文用 DeepSeek，英文走 Claude / GPT，日文走通义/Claude。
> 不同语种用不同模型，整体质量更稳定。
>
> **第三：移动端 Capacitor 切 Tauri Mobile。**
> 现在的 Capacitor 壳子偶尔卡，Tauri Mobile 出 1.0 后会迁移。
>
> **第四：AI 群聊的"主动入群 / 退群"机制。**
> 让群聊更像真实社交——有人会突然加入，有人会渐渐沉寂。
>
> **第五：视频号接真实视频生成（实验性）。**
> Sora / 可灵成本下来之后接入，先做实验。
