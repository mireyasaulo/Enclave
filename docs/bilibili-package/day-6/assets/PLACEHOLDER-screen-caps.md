## 录屏 1：3 个角色的视频号贴（带音）

**对应镜头**：镜头 002（钩子）、镜头 004-006

**录制内容**：
- 第 1 段（10 秒）：进入视频号 tab → 滑到老周的 lo-fi 音乐贴 → 左右滑切换图片 → BGM 自动播 10 秒
- 第 2 段（10 秒）：滑到下一条 → 小宋的古典音乐贴 → BGM 切换明显
- 第 3 段（10 秒）：滑到下一条 → 林薇的电子音乐贴 → BGM 又是不同风格

**⚠️ 关键要求**：**必须保留 BGM 原声**——这一集的核心就是听三种音乐风格的差异

**录制规格**：
- 1080p 30fps 竖屏
- 立体声音频，码率 ≥ 192kbps
- 文件名：`day-6-cap-01-three-bgm.mp4`
