## 关键提示

观点集和演示集的剪辑差异——
1. **多用空镜/海报/概念图**，少用产品截图——避免把观点集做成"广告"
2. **每段之间留 0.5-1 秒静默**，给观众思考空间
3. **重要句子用 1.5x 字号**，比如"你不是大数据的产品。你是自己 AI 世界的主人。"这句要明显放大
4. **BGM 选 cinematic strings 或 post-rock 慢段**，不要用上活泼电子
