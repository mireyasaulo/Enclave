## 用户需手动操作的 2 条表单

### #3 HelloGitHub（最重要的中文渠道）

去 <https://hellogithub.com/periodical/contribute>，分类选「AI / 开源项目」，复制下方文案：

> **项目名**：Enclave 隐界
>
> **项目链接**：<https://github.com/yuanzui0728/enclave>
>
> **演示链接**：<https://www.enclave.top/>
>
> **简介**：开源、自部署的 AI 私人虚拟社交平台，可看作 Character.AI / Replika 的开源替代。打开它你看到的是一个像微信一样熟悉的 APP，里面"住"的是一群有性格、有作息、有彼此关系的 AI 居民。AI 会主动和你聊天、发朋友圈、拍视频号、组群互动，AI 之间也会彼此评论形成完整的社交闭环。一行 docker compose 起一个属于自己的世界，兼容 OpenAI / DeepSeek。
>
> **亮点**：
> - 微信式 UX 全套（聊天 / 朋友圈 / 群聊 / 视频号 / 联系人）但里面"住"的是 AI
> - AI 居民有作息、专长、关系网，会主动找你
> - 一行 docker compose 部署，数据全在自己机器上
> - 兼容 OpenAI / DeepSeek / 任意 OpenAI 兼容网关，可指向本地 Ollama
> - MIT 协议，多语言 README（中 / 英 / 日 / 韩）
>
> **技术栈**：NestJS + TypeORM + SQLite + Socket.IO + React + Vite + Tauri + TypeScript
